## 2. Android SHA Fingerprints

SHA fingerprints are **required** by Firebase for:
- Android app authentication (Google Sign-In)
- Dynamic Links / App Links
- Phone Authentication

They are derived from your **signing keystore**, not from Firebase itself.

### Where to add them in Firebase

> **Firebase Console → Project Settings → General → Your Apps → Android App → Add fingerprint**

Add **both SHA1 and SHA256** for each keystore.

---

### Extracting SHA Fingerprints

#### 🧪 Debug Keystore (for development & testing)

The debug keystore is auto-generated by Android Studio at:
- **macOS/Linux:** `~/.android/debug.keystore`
- **Windows:** `%USERPROFILE%\.android\debug.keystore`

```bash
# macOS / Linux
keytool -list -v \
  -keystore ~/.android/debug.keystore \
  -alias androiddebugkey \
  -storepass android \
  -keypass android

# Windows PowerShell
keytool -list -v `
  -keystore "$env:USERPROFILE\.android\debug.keystore" `
  -alias androiddebugkey `
  -storepass android `
  -keypass android
```

**Expected output (look for these lines):**
```
SHA1:   AB:CD:EF:12:34:56:78:90:AB:CD:EF:12:34:56:78:90:AB:CD:EF:12
SHA256: 12:34:56:78:90:AB:CD:EF:...(64 hex pairs)
```

---

#### 🚀 Release Keystore (for production APK / Play Store)

```bash
# Replace with your actual keystore path and alias
keytool -list -v \
  -keystore /path/to/your-release.jks \
  -alias your-key-alias
```

You will be prompted for the keystore password.

---

#### ⚡ Using Gradle (inside Android Studio project)

```bash
# Debug fingerprint via Gradle
./gradlew signingReport

# Look for the 'release' variant output
```

### Play Store App Signing (Important!)

If you use **Google Play App Signing** (recommended), Google re-signs your APK. You must:

1. Go to **Play Console → Setup → App integrity → App signing key certificate**
2. Copy the **SHA1** and **SHA256** shown there
3. Add them to Firebase as additional fingerprints

## 4. Firestore Security Rules (Recommended)

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /artifacts/{appId}/users/{userId}/{document=**} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}
```
